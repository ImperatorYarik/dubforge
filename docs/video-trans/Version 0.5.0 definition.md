## **Features**

1. Basic transcription without/with English translation
2. Video or Audio translation with voiceover
3. Natural voice over without massive "speedup" or "slow down" issues
4. Export voicover by files with timecodes (separate files with chunks of voice over audio)
5. Eleven Labs integration
6. LLM translation (Separate model from whisper)
7. (SITUATIONAL) LLM context builder (Ain AI agent like system to research video context and provide context for LLM translation agent to avoid miss wording) - Can be ignored if powerfull LLM will translate whole transcription text, not just chunks.
